# FastAPI Sharing

A FastAPI application for sharing files and data.

## Features

- File sharing API
- RESTful endpoints
- Data validation with Pydantic

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
uvicorn main:app --reload
```

## API Endpoints

- GET / - Home page
- POST /upload - Upload file
- GET /files - List files
